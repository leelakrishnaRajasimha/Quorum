## 🚀 Overview

Quorum is an AI Executive Board that helps founders validate startup ideas before pitching to investors.

Instead of receiving feedback from a single AI model, Quorum simulates multiple executive roles working together to evaluate a startup from different business perspectives.

The platform produces a comprehensive board meeting, investment recommendation, startup score, financial analysis, and downloadable executive PDF report.
